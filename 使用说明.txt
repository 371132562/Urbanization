================================================================
 系统快速使用说明
================================================================

欢迎使用！本文档将引导您快速启动和运行本应用程序。

系统要求

- Windows 10 操作系统
- 已成功安装 Docker Desktop

启动与使用步骤

1.  启动 Docker Desktop: 在运行任何脚本之前，请务必先打开并运行您电脑上的 Docker Desktop 应用程序。您可以在系统托盘中看到 Docker 的小鲸鱼图标，表示它正在运行。

2.  运行启动脚本: 请右键点击项目根目录下的 start.ps1 文件，然后选择"使用 PowerShell 运行"选项。
    注意: PowerShell 脚本(.ps1 文件)不能通过双击直接运行，必须通过右键菜单选择运行方式。

3.  等待启动完成: 脚本会自动检查环境、下载最新程序镜像并启动整个应用程序。这个过程可能需要几分钟，请耐心等待。

4.  访问应用: 当脚本窗口显示 "已成功启动！" 的提示时，表示所有服务已准备就绪。您现在可以直接在浏览器中打开以下地址访问应用程序：
    http://localhost:3333

日常管理

- 如何启动服务: 确保 Docker Desktop 正在运行，然后右键点击 start.ps1 并选择"使用 PowerShell 运行"。
- 如何停止服务: 非常简单，直接退出 Docker Desktop 应用程序即可。服务会自动停止。
- 如何更新服务: 当我们发布新版本后，您只需重新运行一次 start.ps1 脚本（右键选择"使用 PowerShell 运行"）。它会自动为您下载并部署最新的版本，无需任何其他操作。

数据管理 (重要)

您的所有数据（如数据库文件等）都保存在 Docker 的一个安全区域（数据卷）中。为了防止数据丢失或迁移数据，我们提供了方便的备份和恢复工具。

如何备份数据

1.  确保 Docker Desktop 正在运行。
2.  右键点击项目根目录下的 backup.ps1 脚本，选择"使用 PowerShell 运行"。
3.  脚本执行成功后，会在项目下创建一个 backups 文件夹，里面包含一个带有时间戳的备份文件（如 urbanization_backup_2024-08-01_12-30-45.tar）。
4.  请妥善保管这个 backups 文件夹和里面的文件，这是您所有数据的快照。建议您定期备份，并将其复制到安全的位置（如U盘、云盘等）。

如何恢复数据

警告：此操作会覆盖当前的所有数据，请谨慎操作！

1.  确保 Docker Desktop 正在运行。
2.  右键点击 restore.ps1 脚本，选择"使用 PowerShell 运行"。
3.  脚本会显示所有可用的备份文件，请输入您想要恢复的备份文件编号。
4.  根据提示，输入 yes 并按回车键确认操作。
5.  恢复成功后，系统会自动重启，您会发现数据已恢复到备份时的状态。

如果您在启动过程中遇到任何[错误]提示，请根据提示检查您的网络连接或 Docker Desktop 是否正常运行。
